== Transaction Tracker

This is a simple application for tracking some basic information of the payment transactions that you make! It'll keep the information of the transactions and generate some useful reports! I wrote this for my personal usage and fun and still working on it! But any kind of contribution is welcome!

It also generates pie chart based on your payments for each category! It gives you the report of your payments in the current month!

Can remind you for monthly payments like bills! And also send you email notifications for some special reports!

And other related functionalities for a payment transaction app!
